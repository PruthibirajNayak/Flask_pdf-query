# Flask_pdf-query

The objective of this project is to build a chatbot capable of answering queries from uploaded PDF, Word, or Excel documents. In this project, I developed a conversational AI system that allows users to upload documents, extract their content, and interact with an AI-powered model to receive contextual responses. The system uses Flask as the backend framework, LlamaIndex/LangChain for document processing, and Ollama for integrating the Large Language Model (LLM). A simple and visually appealing frontend enables users to upload files and interact with the chatbot seamlessly. The chatbot retrieves answers based on the content of the uploaded documents, demonstrating the potential of Natural Language Processing (NLP) and LLMs in building document-based AI assistants. Future improvements may include support for multiple files, enhanced summarization capabilities, and optimized real-time response generation.
